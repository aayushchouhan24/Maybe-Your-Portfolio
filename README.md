# Maybe Your Portfolio

A scroll-driven, cinematic web experience built with GSAP and ScrollTrigger. Designed for learning modern frontend architecture, animation systems, and open source collaboration.

## Features
- Scroll-driven video and animation sync
- Modular architecture (logic, animation, data separation)
- GSAP + ScrollTrigger for advanced effects
- Mobile-aware, responsive design
- Open source contribution workflow

## Setup

1. **Install dependencies**
   ```sh
   npm iX
   ```
2. **Start development server**
   ```sh
   npm run dev
   ```
3. **Build for production**
   ```sh
   npm run build
   ```

## Folder Structure
- `src/` — Main source code
- `public/frames/` — Video/image frames (to be added)
- `.github/` — Contribution templates
